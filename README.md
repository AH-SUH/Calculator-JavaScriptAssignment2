# JavaScript Calculator

This project is a calculator app built with HTML, CSS, and JavaScript.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Clear button
- Dynamic button creation using JavaScript
- DOM manipulation
- forEach loop
- Functions for calculator operations

## Files

- index.html
- style.css
- script.js
